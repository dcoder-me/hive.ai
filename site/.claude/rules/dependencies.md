# Dependencies

## Current Core Stack
- `next`
- `react`
- `react-dom`
- `tailwindcss`
- `eslint`
- `eslint-config-next`

## Preferred Direction
- Prefer built-in Next.js and React capabilities first.
- Prefer small, focused packages with clear need.
- Keep the marketing site lean.

## Avoid
- Do not add Prisma, Supabase, React Query, Zustand, or form libraries unless the site actually needs them.
- Do not add UI kits by default; preserve the site’s own visual system unless a deliberate decision is made.
- Do not add `.js` or `.jsx` files.
